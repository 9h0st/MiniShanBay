# MiniShanBay
##### 这是一个简易的分级阅读程序,基本功能是
1. 打开后看到文章列表
2. 点击列表某项,打开文章
3. 文章界面有一个按钮,点击则在文章中高亮在单词列表中出现的单词
4. 文章内容实现两边对齐
5. 单词实现点击高亮
